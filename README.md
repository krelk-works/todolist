# Todo List App

To-do list enfocada en **deadlines** y **prioridades** para mantener el foco.
Autodetección de idioma (ES/EN).

## Features
- Tareas con deadline ⏳
- Prioridades (Alta/Media/Baja)
- Traducción automática (ES/EN)

## Stack
React + TypeScript + Vite + Tailwind
